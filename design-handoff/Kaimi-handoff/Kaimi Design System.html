Kaimi · the seeker · a BlueMeta product

The design language for an autonomous BD pipeline.

Agents hunt federal opportunities, score them, and draft proposals. Humans triage and supervise. This system gives both halves a shared, trustworthy visual language — built around one organizing idea: two modes that must feel distinct.

Triage

Scanning many opportunities, deciding what to pursue. Fast, scannable, list-driven. Light.

Focus

Supervising one proposal as agents work it. Calm, detailed, status-driven. Dark.

Status vocabulary

The most reused system in the product. It appears on every screen — so it must be unmistakable.

Agent / stage states
Pending
Dormant. Slate, quiet — not yet started.
In Progress
An agent is working. Blue, the dot blinks.
Done
Stage complete. Green, settled.
Needs Human
The loudest. Solid amber, glowing halo, gentle pulse — the system asking for you.
Failed
An agent errored. Red, needs attention.
Bid recommendation
Bid
Pursue. Green — a confident go.
No Bid
Skip. Rose-red — not a fit.
Review
Human judgment needed. Amber — same family as “Needs Human.”
